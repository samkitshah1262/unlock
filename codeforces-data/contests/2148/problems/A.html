A. Sublime Sequence
time limit per test
1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

Farmer John has an integer xx. He creates a sequence of length nn by alternating integers xx and −x−x, starting with xx.

For example, if n=5n=5, the sequence looks like: x,−x,x,−x,xx,−x,x,−x,x.

He asks you to find the sum of all integers in the sequence.

Input

The first line contains an integer tt (1≤t≤1001≤t≤100)  — the number of test cases.

The only line of input for each test case is two integers xx and nn (1≤x,n≤101≤x,n≤10).

Output

For each test case, output the sum of all integers in the sequence.

Example
Input
Copy
4
1 4
2 5
3 6
4 7
Output
Copy
0
2
0
4
