5 seconds

input
standard input
output

We define the humor value of a string tt as the length of the longest palindrome string which occurs at least twice in it.

Formally, a string pp is humor with respect to tt, if and only if both of the following conditions are met:

  1. pp is a palindrome, and
  2. there exist at least two different indices i∈[1,|t|−|p|+1]i∈[1,|t|−|p|+1], such that t[i,i+|p|−1]=pt[i,i+|p|−1]=p.

The humor value of tt is defined as the maximum length among all humor strings with respect to tt.

You are given a string ss of length nn which only contains lowercase Latin letters and qq queries. Each query contains two integers lili, riri, and you need to find the humor value of the string s[li,ri]s[li,ri].

Here, a[l,r]a[l,r] is defined as the string al,al+1,…,aral,al+1,…,ar.

Input

The first line contains two integers nn, qq (1≤n,q≤5⋅1051≤n,q≤5⋅105).

The second line contains a string ss.

The next qq lines contain the description of queries. The ii-th line contains two integers lili, riri (1≤li≤ri≤n1≤li≤ri≤n).

Output

On the ii-th line, output the answer to the ii-th query.

Note

In the second query, one of the longest humor strings with respect to s[2,7]s[2,7] is aa.
