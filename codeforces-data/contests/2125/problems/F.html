F. Timofey and Docker
time limit per test
2 seconds
memory limit per test
512 megabytes
input
standard input
output
standard output

Not so long ago, a certain Timofey learned about docker and now wants to present a report about it at a conference. He has already prepared the text ss.

There are nn people attending the conference. The ii-th attendee will understand Timofey's report if the number of occurrences of the word "docker" as a contiguous substring in the text is not less than lili and not greater than riri.

To ensure that as many people as possible learn about docker, Timofey can change characters in his text.

Help Timofey determine the minimum number of characters that need to be changed so that the topic is understood by the maximum possible number of attendees.

Input

Each test consists of several test cases. The first line contains an integer tt (1≤t≤1041≤t≤104) — the number of test cases. The description of the test cases follows.

The first line of each test case contains one string ss (1≤|s|≤5⋅1051≤|s|≤5⋅105) — Timofey's text, which consists of lowercase Latin letters.

The second line of each test case contains one integer nn (1≤n≤5⋅1051≤n≤5⋅105) — the number of attendees.

The next nn lines of each test case contain two integers li,rili,ri (1≤li≤ri≤1091≤li≤ri≤109).

Additional constraints on the input data:

  • The sum of |s||s| across all test cases does not exceed 5⋅1055⋅105;
  • The sum of nn across all test cases does not exceed 5⋅1055⋅105.
Output

For each test case, print one integer — the minimum number of characters that need to be changed so that the topic is understood by the maximum number of attendees.
