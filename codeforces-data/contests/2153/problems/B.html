B. Bitwise Reversion
time limit per test
1.5 seconds
memory limit per test
256 megabytes
input
standard input
output
standard output

You are given three non-negative integers xx, yy and zz. Determine whether there exist three non-negative integers aa, bb and cc satisfying the following three conditions:

  • a&b=xa&b=x
  • b&c=yb&c=y
  • a&c=za&c=z

where && denotes the bitwise AND operation.

Input

Each test contains multiple test cases. The first line contains the number of test cases tt (1≤t≤1041≤t≤104). The description of the test cases follows.

The first and only line of each test case contains three integers xx, yy and zz (0≤x,y,z≤1090≤x,y,z≤109) — the target values of a&ba&b, b&cb&c and a&ca&c, respectively.

Output

For each test case, output "YES" if there exists three non-negative integers aa, bb, and cc satisfying the above conditions, and "NO" otherwise.

You can output the answer in any case (upper or lower). For example, the strings "yEs", "yes", "Yes", and "YES" will be recognized as positive responses.

Example
Input
Copy
5
1 1 1
3 2 6
4 8 12
9 10 12
12730 3088 28130
Output
Copy
YES
YES
NO
YES
NO
Note

In the first test case, a=3a=3, b=5b=5, and c=9c=9 satisfies the condition as 3&5=13&5=1, 5&9=15&9=1, and 3&9=13&9=1.

In the second test case, a=7a=7, b=3b=3, and c=22c=22 satisfies the condition as 7&3=37&3=3, 3&22=23&22=2, and 7&22=67&22=6.

In the third test case, it can be proven that there are no three non-negative integers aa, bb, and cc such that a&b=4a&b=4, b&c=8b&c=8, and a&c=12a&c=12.
