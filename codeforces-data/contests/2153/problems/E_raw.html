E. Zero Trailing Factorial
time limit per test
3 seconds
memory limit per test
512 megabytes
input
standard input
output
standard output

For all positive integers x≥1x≥1 and k≥2k≥2, let vk(x!)vk(x!) denote the number of trailing zeros in the base-kk representation of x!=x⋅(x−1)⋅…⋅1x!=x⋅(x−1)⋅…⋅1. Formally, vk(x!)vk(x!) is defined as the largest integer ii such that kiki divides x!x!.

For a prime number pp, we can calculate vp(x!)=∑j=1∞⌊xpj⌋vp(x!)=∑j=1∞⌊xpj⌋∗∗. If kk is not prime, write its prime factorization as k=∏peiik=∏piei, where pipi are distinct prime factors and eiei are their corresponding exponents. Then,

vk(x!)=mini⌊vpi(x!)ei⌋.vk(x!)=mini⌊vpi(x!)ei⌋.

wk(a,b)={min(vk(a!),vk(b!))10100if vk(a!)≠vk(b!);otherwise.wk(a,b)={min(vk(a!),vk(b!))if vk(a!)≠vk(b!);10100otherwise.

Next, define fm(a,b)fm(a,b) as the minimum weight of the pair (a,b)(a,b) with respect to kk, taken over all integers kk with 2≤k≤m2≤k≤m:

fm(a,b)=min2≤k≤mwk(a,b).fm(a,b)=min2≤k≤mwk(a,b).

You are given two integers nn and mm. Your task is to compute the sum of fm(x,n)fm(x,n) over all positive integers xx less than nn:

∑1≤x≤n−1fm(x,n).∑1≤x≤n−1fm(x,n).

It can be shown that under the given constraints, the result is strictly less than 1010010100.

∗∗⌊y⌋⌊y⌋ denotes the floor of yy, which is the greatest integer less than or equal to yy.

Input

Each test contains multiple test cases. The first line contains the number of test cases tt (1≤t≤1001≤t≤100). The description of the test cases follows.

The first and only line of each test case contains two integers nn and mm (2≤n≤m≤1072≤n≤m≤107) — the parameters of the function.

Note that there are no constraints on the sum of n,mn,m over all test cases.

Output

For each test case, output a single integer representing ∑1≤x≤n−1fm(x,n)∑1≤x≤n−1fm(x,n).
