F1. Serval and Colorful Array (Easy Version)
time limit per test
2 seconds
memory limit per test
512 megabytes
input
standard input
output
standard output

This is the easy version of the problem. The difference between the versions is that in this version, n≤3000n≤3000. You can hack only if you solved all versions of this problem.

Serval has a magic number kk (k≥2k≥2). We call an array rr colorful if and only if:

  • The length of rr is kk, and
  • Each integer between 11 and kk appears exactly once in rr.

You are given an array aa consisting of nn integers between 11 and kk. It is guaranteed that each integer between 11 and kk appears in aa at least once. You can perform the following operation on aa:

  • Choose an index ii (1≤i<n1≤i<n), then swap aiai and ai+1ai+1.

Find the minimum number of operations needed to make at least one subarray∗∗ of aa colorful. It can be shown that this is always possible under the constraints of the problem.

∗∗An array bb is a subarray of an array aa if bb can be obtained from aa by the deletion of several (possibly, zero or all) elements from the beginning and several (possibly, zero or all) elements from the end.

Input

Each test contains multiple test cases. The first line contains the number of test cases tt (1≤t≤10001≤t≤1000). The description of the test cases follows.

The first line of each test case contains two integers nn and kk (2≤k≤n≤30002≤k≤n≤3000) — the length of the array aa and Serval's magic number.

The second line contains nn integers a1,a2,…,ana1,a2,…,an (1≤ai≤k1≤ai≤k) — the elements of the array aa. It is guaranteed that each integer between 11 and kk appears in aa at least once.

It is guaranteed that the sum of nn over all test cases does not exceed 30003000.

Output

For each test case, output a single integer — the minimum number of operations needed to make at least one subarray of aa colorful.

Note

In the first test case, since the subarrays [a1,a2]=[1,2][a1,a2]=[1,2] and [a2,a3]=[2,1][a2,a3]=[2,1] are already colorful, we do not need to perform any operations. Thus, the answer is 00.

In the second test case, we can swap a1a1 and a2a2 to obtain [1,2,1,3_,1,1,2][1,2,1,3––––––,1,1,2], which has a colorful subarray [a2,a3,a4]=[2,1,3][a2,a3,a4]=[2,1,3]. And the given array initially does not have any colorful subarrays, so the answer is 11.
