A. Serval and String Theory

1 second
memory limit per test

input
standard input

A string rr consisting only of lowercase Latin letters is called universal if and only if rr is lexicographically smaller∗∗ than the reversal†† of rr.

You are given a string ss consisting of nn lowercase Latin letters. You are required to make ss universal. To achieve this, you can perform the following operation on ss at most kk times:

  • Choose two indices ii and jj (1≤i,j≤n1≤i,j≤n), then swap sisi and sjsj. Note that if i=ji=j, you do nothing.

Determine whether you can make ss universal by performing the above operation at most kk times.

∗∗A string aa is lexicographically smaller than a string bb of the same length, if and only if the following holds:

  • in the first position where aa and bb differ, the string aa has a letter that appears earlier in the alphabet than the corresponding letter in bb.

††The reversal of a string rr is the string obtained by writing rr from right to left. For example, the reversal of the string abcadabcad is dacbadacba.

Input

Each test contains multiple test cases. The first line contains the number of test cases tt (1≤t≤5001≤t≤500). The description of the test cases follows.

The first line of each test case contains two integers nn and kk (1≤n≤1001≤n≤100, 0≤k≤1040≤k≤104) — the length of the string ss, and the maximum number of operations you can perform.

The second line contains a string ss consisting of nn lowercase Latin letters.

Output

For each test case, print "YES" if it is possible to make ss universal by performing the operation at most kk times. Otherwise, print "NO".

You can output the answer in any case (upper or lower). For example, the strings "yEs", "yes", "Yes", and "YES" will be recognized as positive responses.

Note

In the first test case, ss will keep the same after any operations. However, the reversal of aa is still aa, so it is impossible to make ss universal.

In the second test case, the string revrev is lexicographically smaller than verver. Thus, ss is already universal.

In the fifth test case, you can perform the operations as follows:

  1. Swap s4s4 and s7s7. After this operation, ss becomes uniservaluniserval;
  2. Swap s1s1 and s3s3. After this operation, ss becomes inuservalinuserval.

And the string inuservalinuserval is universal.
