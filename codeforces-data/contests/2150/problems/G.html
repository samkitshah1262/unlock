You are given three positive integers xx, yy, and kk.

You are also given a binary string∗∗ aa (|a|=x+y|a|=x+y).

Count the number of binary strings bb, modulo 998244353998244353, such that:

  • there are exactly xx 00 in bb.
  • there are exactly yy 11 in bb.
  • there exists an integer ii (1≤i≤x+y−11≤i≤x+y−1) such that min(f(b1b2…bi),f(bi+1bi+2…bx+y))≥kmin(f(b1b2…bi),f(bi+1bi+2…bx+y))≥k, where f(s)f(s) gives the length of the longest non-decreasing subsequence†† in ss.
  • bb is lexicographically larger‡‡ than aa.

∗∗A binary string is a string that only consists of characters 00 and 11.

††A sequence aa is a subsequence of a sequence bb if aa can be obtained from bb by the deletion of several (possibly, zero or all) elements. For example, subsequences of 10111011011101 are 00, 11, 1111111111, 01110111, but not 000000 nor 1110011100.

‡‡A string pp is lexicographically larger than another string qq if and only if one of the following holds:

  • qq is a prefix of pp, but p≠qp≠q; or
  • in the first position where pp and qq differ, the string pp has a larger element than the corresponding element in qq.
Input

Each test contains multiple test cases. The first line contains the number of test cases tt (1≤t≤50001≤t≤5000). The description of the test cases follows.

The first line of each test case contains three integers xx, yy, and kk (1≤x,y≤50001≤x,y≤5000, 1≤k<x+y1≤k<x+y).

The second line of each test case contains a binary string aa (|a|=x+y|a|=x+y) consisting of characters 00 and 11.

It is guaranteed that neither the sum of xx nor the sum of yy over all test cases exceeds 50005000.

Output

For each test case, output an integer on a new line, the number of binary strings that satisfy the conditions modulo 998244353998244353.
