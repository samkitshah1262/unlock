This is an interactive problem.

There are two hidden non-negative integers xx and yy (0≤x,y<2300≤x,y<230). You can ask no more than 22 queries of the following form.

  • Pick a non-negative integer nn (0≤n<2300≤n<230). The judge will respond with the value of (n|x)+(n|y)(n|x)+(n|y), where || denotes the bitwise OR operation.

After this, the judge will give you another non-negative integer mm (0≤m<2300≤m<230). You must answer the correct value of (m|x)+(m|y)(m|x)+(m|y).

Input

Each test contains multiple test cases. The first line contains the number of test cases tt (1≤t≤1041≤t≤104). The description of the test cases follows.

Interaction

Two hidden integers xx and yy are chosen (0≤x,y<2300≤x,y<230). Note that xx and yy might be different for different test cases.

The interactor in this task is not adaptive. In other words, the integers xx and yy do not change during the interaction.

To ask a query, pick an integer nn (0≤n<2300≤n<230) and print only the integer nn to a line.

You will receive a single integer, the value of (n|x)+(n|y)(n|x)+(n|y).

You may make no more than 22 queries of the following form.

After you finish your queries, output "!" in a line. You will receive an integer mm (0≤m<2300≤m<230). Note that the value of mm is also fixed before interaction.

You must output only the value of (m|x)+(m|y)(m|x)+(m|y) in a line. Note that this line is not considered a query and is not taken into account when counting the number of queries asked.

After this, proceed to the next test case.

If you make more than 22 queries during an interaction, your program must terminate immediately, and you will receive the Wrong Answer verdict. Otherwise, you can get an arbitrary verdict because your solution will continue to read from a closed stream.

After printing a query do not forget to output the end of line and flush the output. Otherwise, you will get Idleness limit exceeded. To do this, use:

  • fflush(stdout) or cout.flush() in C++;
  • System.out.flush() in Java;
  • flush(output) in Pascal;
  • stdout.flush() in Python;
  • see the documentation for other languages.
