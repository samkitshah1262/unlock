F. Weapon Upgrade
time limit per test
6 seconds
memory limit per test
512 megabytes
input
standard input
output
standard output

Monocarp is playing a game. The game is divided into stages, and there are a total of 13371337 stages. The game character uses a weapon with two attack modes: physical and elemental. Initially, the weapon's physical attack power is 11, and its elemental attack power is also 11.

During the ii-th stage of the game, the following events occur in order:

  1. If i≤ni≤n, a monster with a physical defense aiai and elemental defense bibi appears. Otherwise, nothing happens.
  2. Monocarp can shoot at exactly one of the remaining monsters. If the monster's physical defense is at most the character's physical attack power or the monster's elemental defense is at most the character's elemental attack power, the monster will be killed. Otherwise, nothing happens.
  3. Each surviving monster deals 11 damage to the game character.

After each kill, Monocarp can upgrade one of the attacks of the weapon by 11.

Your task is to calculate the minimum damage that Monocarp can get in order to kill all the monsters in the game (or report that it is impossible to kill all the monsters).

Input

The first line contains a single integer nn (1≤n≤5001≤n≤500).

The second line contains nn integers a1,a2,…,ana1,a2,…,an (1≤ai≤5001≤ai≤500).

The third line contains nn integers b1,b2,…,bnb1,b2,…,bn (1≤bi≤5001≤bi≤500).

Output

Print a single integer — the minimum damage that Monocarp can get in order to kill all the monsters in the game; if it is impossible to kill all the monsters, print -1.

Note

In the first example, Monocarp can proceed as follows:

11-st stage:

  • 11-st monster appears;
  • Monocarp cannot kill any monster;
  • 11 alive monster deals 11 damage, so the total damage Monocarp got is 11.
22-nd stage:
  • 22-nd monster appears;
  • Monocarp kills 22-nd monster (using elemental attack) and upgrades physical damage;
  • 11 alive monster deals 11 damage, so the total damage Monocarp got is 22.
33-rd stage:
  • 33-rd monster appears;
  • Monocarp kills 11-st monster (using physical attack) and upgrades elemental damage;
  • 11 alive monster deals 11 damage, so the total damage Monocarp got is 33.
44-th stage:
  • 44-th monster appears;
  • Monocarp kills 44-th monster (using elemental attack) and upgrades elemental damage;
  • 11 alive monster deals 11 damage, so the total damage Monocarp got is 44.
55-th stage:
  • Monocarp kills 33-rd monster (using elemental attack) and upgrades physical damage;
  • all monsters are killed and the total damage Monocarp got is 44.
