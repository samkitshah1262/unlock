C. Manhattan Pairs
time limit per test
2 seconds
memory limit per test
256 megabytes
input
standard input
output
standard output

You are given nn points (xi,yi)(xi,yi) on a 2D plane, where nn is even. Select n2n2 disjoint pairs (ai,bi)(ai,bi) to maximize the sum of Manhattan distances between points in pairs. In other words, maximize

∑i=1n/2|xai−xbi|+|yai−ybi|.∑i=1n/2|xai−xbi|+|yai−ybi|.

Input

Each test contains multiple test cases. The first line contains the number of test cases tt (1≤t≤1041≤t≤104). The description of the test cases follows.

The first line of each test case contains a single even integer nn (2≤n≤2⋅1052≤n≤2⋅105) — the number of points.

The ii-th of the next nn lines contains two integers xixi and yiyi (−106≤xi,yi≤106−106≤xi,yi≤106) — the coordinates of the ii-th point.

It is guaranteed that the sum of nn over all test cases does not exceed 2⋅1052⋅105.

Output

For each test case, output n2n2 lines, the ii-th line containing two integers aiai and bibi — the indices of points in the ii-th pair.

If there are multiple solutions, print any of them.

Note

In the first test case, an optimal solution is to select the pairs (1,4)(1,4) and (2,3)(2,3), which achieves a distance sum of 5+3=85+3=8.

In the second test case, an optimal solution is to select the pairs (1,8)(1,8), (9,10)(9,10), (5,7)(5,7), (2,3)(2,3), (4,6)(4,6), which achieves a distance sum of 4+7+10+5+7=334+7+10+5+7=33.
