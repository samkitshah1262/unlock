B. Games
time limit per test
2 seconds
memory limit per test
256 megabytes
input
standard input
output
standard output

Alice and Bob are planning to play an online game together but haven't decided on which one yet. Alice has a list of nn games she enjoys: a1,a2,…,ana1,a2,…,an. Bob, on the other hand, has a list of mm games he likes: b1,b2,…,bmb1,b2,…,bm. They share at least one game in common between their lists.

To choose a game, they take turns suggesting games from their lists. Alice begins by suggesting one of her favorite games. If Bob likes it, they play that game. If not, Bob suggests one of his favorite games. If Alice likes it, they play that game. This alternating process continues, with Alice and Bob each suggesting games from their lists in turn, ensuring that no game is suggested more than once.

Your task is to calculate the maximum possible number of suggestions they will make while choosing which game to play.

Input

The first line contains a single integer tt (1≤t≤10001≤t≤1000) — the number of test cases.

The first line of each test case contains two integers nn and mm (1≤n,m≤1001≤n,m≤100).

The second line contains nn integers a1<a2<⋯<ana1<a2<⋯<an (1≤ai≤1001≤ai≤100).

The third line contains mm integers b1<b2<⋯<bmb1<b2<⋯<bm (1≤bi≤1001≤bi≤100).

Arrays aa and bb contain at least one common integer.

Output

For each test case, print a single integer — the maximum possible number of suggestions they will make while choosing which game to play.

Note

In the first test case, the maximum number of suggestions is 33. It is achieved as follows:

  • Alice suggests game 11, but Bob does not like it;
  • Bob suggests game 55, but Alice does not like it;
  • Alice suggests game 22, which Bob likes, and they go to play that game.

In the second test case, Alice can only suggest game 55, which Bob likes, so they will immediately go to play it.

In the third test case, the maximum number of suggestions is 44. It is achieved as follows:

  • Alice suggests game 77, but Bob does not like it;
  • Bob suggests game 66, but Alice does not like it;
  • Alice suggests game 11, but Bob does not like it;
  • Bob suggests game 44, which Alice likes, and they go to play it.
