E. Perfect Cut
time limit per test
2 seconds
memory limit per test
256 megabytes
input
standard input
output
standard output

You are given a string ss consisting of characters 0, 1, and/or ?.

Let's call a binary (consisting of 0 and/or 1) string perfect if you can cut the string into two non-empty parts: a prefix aa and the suffix bb, such that ai≥biai≥bi for each ii from 11 to min(|a|,|b|)min(|a|,|b|).

Your task is to calculate the number of ways to replace all the question marks (independently) with zeros and ones so that the resulting string is perfect mod. Two ways are considered different if there exists at least one position that has different characters in these two ways. Since the answer can be large, print it modulo 998244353998244353.

Input

The first line contains a single integer tt (1≤t≤1041≤t≤104) — the number of test cases.

The only line of each test case contains a string ss (2≤|s|≤2⋅1052≤|s|≤2⋅105), consisting of characters 0, 1, and/or ?.

Additional constraint on the input: the sum of the lengths of the strings ss over all test cases doesn't exceed 2⋅1052⋅105.

Output

For each test case, print a single integer — the number of ways to replace all the question marks (independently) with zeros and ones so that the resulting string is perfect, taken modulo 998244353998244353.

Note

In the first test case, the string 001 can be split into a prefix of length 11 and a suffix of length 22.

In the fourth test case, the following strings can be obtained:

  • The string 11010, which can be split into a prefix of length 22 and a suffix of length 33;
  • The string 11000, which can be split into a prefix of length 44 and a suffix of length 11.
