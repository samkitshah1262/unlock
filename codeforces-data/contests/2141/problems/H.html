H. Merging Vertices in a Graph
time limit per test
3 seconds
memory limit per test
512 megabytes
input
standard input
output
standard output

You are given an undirected graph, initially containing nn vertices and mm edges. You can perform the following operation on this graph:

  • Choose any two distinct vertices of the graph, remove them, and insert a new vertex that is connected by edges to all vertices that were connected to both of the chosen vertices. That is, if the chosen vertices are uu and vv, and the new vertex is xx, then edges (x,y)(x,y) are added to the graph for all such vertices yy that both edges (u,y)(u,y) and (v,y)(v,y) existed before the operation.

This operation can be performed any number of times until there exists a vertex in the graph that is directly connected by edges to all other vertices. As soon as such a vertex appears, the process ends. Additionally, if such a vertex already exists in the graph initially, no operations can be performed.

Your task is to count the minimum and maximum number of operations that you can perform.

Input

The first line contains two integers nn and mm (1≤n≤2⋅1051≤n≤2⋅105; 0≤m≤min(n(n−1)2,2⋅105)0≤m≤min(n(n−1)2,2⋅105)).

The ii-th of the following mm lines contains two integers xi,yixi,yi (1≤xi,yi≤n1≤xi,yi≤n; xi≠yixi≠yi) — the endpoints of the ii-th edge. There is at most one edge between each pair of vertices.

Output

Output two integers — the minimum and the maximum number of operations.

Note

In the first example, the shortest sequence of operations is as follows:

  • Choose vertices 11 and 33, then the resulting vertex will be connected to vertices 22, 44, 55, and there are no other vertices in the graph.

The longest sequence of operations is as follows:

  • Choose vertices 11 and 55. Let the resulting vertex be 66; it will not be connected to any of the remaining vertices.
  • Choose vertices 22 and 44. Let the resulting vertex be 77; it will be connected to vertex 33.
  • Choose vertices 77 and 33. Let the resulting vertex be 88; it will not be connected to any of the remaining vertices.
  • Choose vertices 66 and 88. Only one vertex will remain in the graph, so it will be connected to all other vertices.

In the second example, there is already a vertex in the graph that is connected to all others, so no operations can be performed.

In the third example, it will take 199999199999 operations to leave only one vertex in the graph, regardless of the approach.
