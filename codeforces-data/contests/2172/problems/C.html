C. Circles Are Far from Each Other

You are given nn circles and two integer parameters kk and ℓℓ. The ii-th circle has radius riri and ri≥rjri≥rj holds for every 1≤i<j≤n1≤i<j≤n. The task is to draw these nn circles on a 2D plane so that the following conditions are simultaneously satisfied. Before proceeding to the conditions, let us recall that:

  • Each circle is uniquely determined by its center OO and radius rr.
  • A circle of radius rr is defined as the set of all points whose distance from its center OO is exactly rr. All distances considered in this task are Euclidean.
  • The interior region of a circle is defined as the set of all points whose distance from its center OO is less than rr.
  • We say that one circle CC encloses another circle C′C′ if all points of C′C′ lie inside the interior region of CC.

You need to draw these nn circles to satisfy all of the following conditions:

  1. The centers of all circles are collinear.
  2. The distance between any two centers is at most kk.
  3. No two circles intersect.
  4. If a circle encloses two circles CC and C′C′, then either CC encloses C′C′ or C′C′ encloses CC.

  5. The first ℓℓ circles may or may not be enclosed by other circles, whereas each of the remaining n−ℓn−ℓ circles must be enclosed by at least one circle.

An arrangement that satisfies all the above conditions is called feasible. For a feasible arrangement AA, define its quality d(A)d(A) as the minimum distance between any two points belonging to different circles.

If there exists at least one feasible arrangement for the given case, output the maximum possible quality among all feasible arrangements. If no feasible arrangements exist, output 00.

Input

The first line contains three integers kk, nn, and ℓℓ, representing the maximum distance between centers, the number of circles to be drawn, and the number of circles that may or may not be enclosed by other circles, respectively.

The second line contains nn integers r1,r2,…,rnr1,r2,…,rn, where riri is the radius of the ii-th circle.

  • 1≤k≤1091≤k≤109
  • 2≤n≤1052≤n≤105
  • 1≤ℓ≤min{n,200}1≤ℓ≤min{n,200}
  • 1≤ri≤1091≤ri≤109
  • ri≥rjri≥rj for every 1≤i<j≤n1≤i<j≤n.
Output

If no feasible arrangements exist, output a single integer 00.

Otherwise, output the maximum possible quality d(A)d(A) among all feasible arrangements AA in the following format:

  • If d(A)d(A) is an integer, output it as a single integer.
  • If d(A)d(A) is not an integer, output it in the rational form a/ba/b such that 1≤a,b≤1091≤a,b≤109, gcd(a,b)=1gcd(a,b)=1, and |d(A)−a/b||d(A)−a/b| is minimized. If multiple such forms satisfy the constraints, print any.
