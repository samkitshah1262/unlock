F. Hot Matrix
time limit per test
4 seconds
memory limit per test
512 megabytes
input
standard input
output
standard output

Piggy Zhou loves matrices, especially those that make him get excited, called hot matrix.

A hot matrix of size n×nn×n can be defined as follows. Let ai,jai,j denote the element in the ii-th row, jj-th column (1≤i,j≤n1≤i,j≤n).

  1. Each column and row of the matrix is a permutation of all numbers from 00 to n−1n−1.
  2. For each pair of indices ii, jj, such that 1≤i,j≤n1≤i,j≤n, ai,j+ai,n−j+1=n−1ai,j+ai,n−j+1=n−1.
  3. For each pair of indices ii, jj, such that 1≤i,j≤n1≤i,j≤n, ai,j+an−i+1,j=n−1ai,j+an−i+1,j=n−1.
  4. All ordered pairs (ai,j,ai,j+1)(ai,j,ai,j+1), where 1≤i≤n1≤i≤n, 1≤j<n1≤j<n, are distinct.
  5. All ordered pairs (ai,j,ai+1,j)(ai,j,ai+1,j), where 1≤i<n1≤i<n, 1≤j≤n1≤j≤n, are distinct.

Now, Piggy Zhou gives you a number nn, and you need to provide him with a hot matrix if the hot matrix exists for the given nn, or inform him that he will never get excited if the hot matrix does not exist for the given nn.

Input

Each test contains multiple test cases. The first line contains the number of test cases tt (1≤t≤10001≤t≤1000). The description of the test cases follows.

The only line of each test case contains one integer nn (1≤n≤30001≤n≤3000).

It is guaranteed that the sum of nn across all test cases does not exceed 30003000.

Output

For each test case, print "NO" (without quotes) if the hot matrix does not exist for the given nn.

Otherwise, print "YES" (without quotes) on the first line. Then output nn rows, with nn numbers in each row, representing the hot matrix of size n×nn×n that meets the requirements of the problem.

If there are multiple solutions, print any of them.

Note

In the first, second and fourth test case, it can be verified that the matrix provided in the example meets all the conditions stated in the problem.

In the third test case, by enumerating all possible matrices, it can be proven that there is no hot matrix that satisfies the conditions of the problem.
