This is the easy version of the problem. The difference between the versions is that in this version, the limits on nn and the time limit are smaller. You can hack only if you solved all versions of this problem.

You are given an integer nn, and you need to compute (∑nk=1kmodφ(k))mod232(∑k=1nkmodφ(k))mod232, where φ(k)φ(k) equals the number of positive integers no greater than kk that are coprime with kk.

Input

The only line contains a single integer nn (1≤n≤1010)1≤n≤1010).

Output

Print a single integer, representing (∑nk=1kmodφ(k))mod232(∑k=1nkmodφ(k))mod232.
