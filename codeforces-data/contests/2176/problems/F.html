F. Omega Numbers

For a given number nn, consider the function ω(n)ω(n), which is equal to the number of unique prime numbers in the prime factorization of the number nn.

For example, ω(12)=ω(22⋅3)=2ω(12)=ω(22⋅3)=2. And ω(120)=ω(23⋅3⋅5)=3ω(120)=ω(23⋅3⋅5)=3.

For an array of natural numbers aa and a natural number kk, we define f(a,k)=∑i<jω(ai⋅aj)kf⁡(a,k)=∑i<jω(ai⋅aj)k for all i<ji<j.

You are given an array of natural numbers aa of length nn and a natural number kk. Calculate f(a,k)f⁡(a,k) modulo 998244353998244353.

Input

Each test contains multiple test cases. The first line contains the number of test cases tt (1≤t≤1041≤t≤104). The description of the test cases follows.

The first line of each test case contains 2 integers nn and kk (1≤n≤2⋅105,1≤k≤1091≤n≤2⋅105,1≤k≤109) —the length of the array aa and the exponent of the operation, respectively.

The second line of each test case contains nn natural numbers a1,a2,…,ana1,a2,…,an (1≤ai≤n1≤ai≤n) —the array aa.

It is guaranteed that the sum of nn across all test cases does not exceed 2⋅1052⋅105.

Output

For each test case, output a single line containing an integer — the value of the function f(a,k)f⁡(a,k) modulo 998244353998244353.
