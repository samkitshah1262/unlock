Teto is playing the hit rhythm game osu!. The game can be described by a binary string∗∗ ss of length nn and a positive integer kk where the following will happen in order:

  • You will choose some positions in ss to protect.
  • Then for each ii (1≤i≤n1≤i≤n) in increasing order, Teto can set sisi to 00 if all the following are true:
    • si=1si=1,
    • sisi is not protected,
    • the previous k−1k−1 elements do not contain 11. More formally, 11 does not occur in smax(1,i−k+1),…,si−1smax(1,i−k+1),…,si−1.

You dislike Teto (for some reason). So determine the minimum number of positions you need to protect to force her to leave ss unchanged.

∗∗A binary string is a string that only consists of characters 00 and 11.

Input

Each test contains multiple test cases. The first line contains the number of test cases tt (1≤t≤1001≤t≤100). The description of the test cases follows.

The first line of each testcase contains integers nn and kk (2≤n≤10002≤n≤1000; 2≤k≤n2≤k≤n) — the length of ss and kk.

The second line of each test case contains a binary string ss of length nn consisting of characters 00 and 11.

The sum of nn across all testcases does not exceed 10001000.

Output

For each testcase, output the minimum number of positions you need to protect to force Teto to leave the string unchanged.
