D. Scammy Game Ad

3 seconds

256 megabytes
input
standard input

Consider the following game.

In this game, a level consists of nn pairs of gates. Each pair contains one left gate and one right gate. Each gate performs one of two operations:

  • Addition Operation (+ a): Increases the number of people in a lane by a constant amount aa.
  • Multiplication Operation (x a): Multiplies the current number of people in a lane by an integer aa. This means the number of people increases by (a−1)(a−1) times the current count in that lane.

The additional people gained from each operation can be assigned to either lane. However, people already in a lane cannot be moved to the other lane.

Initially, there is one person in each lane. Your task is to determine the maximum total number of people that can be achieved by the end of the level.

Input

The first line contains an integer tt (1≤t≤1041≤t≤104) — the number of test cases.

The first line of each test case contains one integer nn (1≤n≤301≤n≤30) — the number of pairs of gates.

The next nn lines of each test case provide the information for the left gate followed by the right gate of each gate pair. The information for each gate is given in the form + aa (1≤a≤10001≤a≤1000) or x aa (2≤a≤32≤a≤3) for some integer aa.

Output

For each test case, output a single integer — the maximum total number of people at the end of the level.

Initially, we have l=1l=1 person in the left lane and r=1r=1 person in the right lane.

After passing through the first pair of gates, we gain 44 people from the left gate and 1⋅(2−1)=11⋅(2−1)=1 person from the right gate, for a total of 4+1=54+1=5 people. We allocate 22 people to the left lane and 33 people to the right lane. This results in l=1+2=3l=1+2=3 people in the left lane and r=1+3=4r=1+3=4 people in the right lane.

After passing through the second pair of gates, we gain 3⋅(3−1)=63⋅(3−1)=6 people from the left gate and 4⋅(3−1)=84⋅(3−1)=8 people from the right gate, for a total of 6+8=146+8=14 people.
