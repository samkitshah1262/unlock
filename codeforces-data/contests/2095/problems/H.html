H. Blurry Vision
time limit per test
1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

Please direct your vision to line number xx.

Now, read the word you see on that line.

If you struggle, I might have to prescribe eyeglasses.

Input

The only line contains one integer xx (1≤x≤111≤x≤11) — the line number.

Output

Output the word on line number xx.

Example
Input
Copy
1
Output
Copy
CODEFORCES
Note

If you fail to read the first line, even glasses won't save you.
