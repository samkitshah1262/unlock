Let's define a value of a string, consisting of letters "XVI", as follows:

  • the value of 'X' is 1010;
  • the value of 'V' is 55;
  • the value of 'I' is 11 or −1−1: −1−1 if the next position contains a letter 'X' or 'V'; 11 otherwise;
  • the value of the entire string is the sum of the values of all letters in it.

You are given a string of characters "XVI?" and asked qq queries about it.

In the ii-th query, three integers are provided:

  • cX cV cIcX cV cI — the number of available letters 'X', 'V', and 'I', respectively.

What is the minimum value of the string that can be obtained if all question marks are replaced with the letters 'X', 'V', 'I' so that the number of used letters does not exceed the number of available letters of each type?

Input

The first line contains one integer tt (1≤t≤1041≤t≤104) — the number of test cases.

The first line of each test case contains two integers nn and qq (1≤n,q≤3⋅1051≤n,q≤3⋅105) — the length of the string and the number of queries, respectively.

The second line contains a string consisting of nn characters 'X', 'V', 'I' and/or '?'.

The ii-th of the following qq lines contains three integers cX,cVcX,cV, and cIcI (0≤cX,cV,cI≤n0≤cX,cV,cI≤n) — the number of available letters 'X', 'V' and 'I' in the ii-th query.

Additional constraints on the input:

  • the sum of nn over all test cases does not exceed 3⋅1053⋅105;
  • the sum of qq over all test cases does not exceed 3⋅1053⋅105;
  • cX+cV+cIcX+cV+cI is greater than or equal to the number of '?' characters in the given string.
Output

For each query, print a single integer — the minimum value of the string that can be obtained by replacing all question marks with the available letters 'X', 'V', and/or 'I'.
