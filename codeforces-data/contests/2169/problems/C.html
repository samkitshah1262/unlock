C. Range Operation
time limit per test
2 seconds
memory limit per test
512 megabytes
input
standard input
output
standard output

You are given an integer array aa of length nn.

You can perform the following operation: choose a range [l,r][l,r] (1≤l≤r≤n1≤l≤r≤n) and replace the value of elements al,al+1,…,aral,al+1,…,ar with (l+r)(l+r).

Your task is to calculate the maximum possible total array sum if you can perform the aforementioned operation at most once.

Input

The first line contains a single integer tt (1≤t≤1041≤t≤104) — the number of test cases.

The first line of each test case contains a single integer nn (1≤n≤2⋅1051≤n≤2⋅105).

The second line contains nn integers a1,a2,…,ana1,a2,…,an (0≤ai≤2n0≤ai≤2n).

Additional constraint on the input: the sum of nn over all test cases doesn't exceed 2⋅1052⋅105.

Output

For each test case, print a single integer — the maximum possible total array sum if you can perform the aforementioned operation at most once.

Note

In the first example, you can perform the operation on the subarray [3,3][3,3], resulting in the array [2,5,6][2,5,6] and the sum 1313.

In the second example, you don't need to perform any operation.

In the third example, you can perform the operation on the subarray [1,4][1,4], resulting in the array [5,5,5,5][5,5,5,5] and the sum 2020.

In the fourth example, you can perform the operation on the subarray [2,3][2,3], resulting in the array [3,5,5,9,10][3,5,5,9,10] and the sum 3232.
