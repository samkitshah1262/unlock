D. Simple Permutation
time limit per test
2 seconds
memory limit per test
256 megabytes
input
standard input
output
standard output

Given an integer n. Construct a permutation p1,p2,…,pn of length n that satisfies the following property:

For 1≤i≤n, define ci=⌈p1+p2+…+pii⌉, then among c1,c2,…,cn there must be at least ⌊n3⌋−1 prime numbers.

Input

The first line contains an integer t (1≤t≤10) — the number of test cases. The description of the test cases follows.

In a single line of each test case, there is a single integer n (2≤n≤105) — the size of the permutation.

Output

For each test case, output the permutation p1,p2,…,pn of length n that satisfies the condition. It is guaranteed that such a permutation always exists.

Example
Input
Copy
3
2
3
5
Output
Copy
2 1
2 1 3
2 1 3 4 5
Note

In the first test case, c1=⌈21⌉=2, c2=⌈2+12⌉=2. Both are prime numbers.

In the third test case, c1=⌈21⌉=2, c2=⌈32⌉=2, c3=⌈63⌉=2, c4=⌈104⌉=3, c5=⌈155⌉=3. All these numbers are prime.
